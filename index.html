Count the Notes: Histogram-Based Supervision for Automatic Music Transcription

Jonathan Yaffe1, Ben Maman1, Meinard Müller2 Amit H. Bermano1
1Tel Aviv University, 2AudioLabs Erlangen
Paper arXiv Code & Data

Abstract

CountEM is a novel framework for Automatic Music Transcription that uses note histograms rather than aligned data. It removes the need for temporal alignment by relying on onset counts within wide time windows. The model iteratively refines its predictions through Expectation-Maximization, significantly reducing annotation effort while maintaining strong performance across diverse datasets.
